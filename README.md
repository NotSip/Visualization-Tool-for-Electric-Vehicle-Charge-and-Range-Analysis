# India EV Market Intelligence Portal:
A Full-Stack Data Science Project: MySQL → Tableau → Flask

# Project Overview
This project provides a comprehensive analysis of the Electric Vehicle (EV) landscape in India. By integrating disparate datasets, I identified the critical "Market Gap" where global brands fail and local manufacturers like Tata and MG succeed.

## Key Insight
While the global market features **33 major brands**, the Indian ecosystem is currently dominated by **9 core manufacturers**. This project visualizes the correlation between "Range per Lakh" and consumer adoption.

##  Tech Stack
* **Database:** MySQL (Advanced querying, View creation, Data cleaning)
* **Visualization:** Tableau Public (Interactive Storytelling, Geospatial mapping)
* **Backend:** Flask (Python-based web routing)
* **Frontend:** HTML5, Bootstrap 5 (Responsive UI/UX)

## Key Features
* **The EV Story:** An interactive Tableau Story that guides users through the evolution of the Indian EV market.
* **Infrastructure Heatmap:** Visualising charging station density vs. regional sales.
* **Price-to-Range Scatter Plot:** Identifying the "Sweet Spot" for mass-market adoption.
* **Clean Web Interface:** A modern dashboard accessible via any browser.
